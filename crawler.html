Crawler¶

Warning

The SecCrawler class will be deprecated in v0.2.0 in favor of the newly released Filing class (found in SECEdgar.filings). See the complete list, reference filings.

The SECEdgar module is largely centered around the SecCrawler class. This class fetches the filing information.

class SECEdgar.crawler.SecCrawler(data_path='/home/travis/build/coyo8/sec-edgar/SEC-Edgar-Data')¶

Main crawler object for SEC filings.

Parameters

data_path (str) – Path where data will be saved.

New in version 0.1.4.

The crawler has many different methods to fetch different filings from the Edgar database. Each method will create new documents in the specified data path.

SecCrawler.filing_10Q(company_code, cik, priorto, count)¶

Fetch 10Q reports before priorto date.

Parameters

  • company_code (str) – Code used to help find company filings. Often the company’s ticker is used.

  • cik (Union[str, int]) – Central Index Key assigned by SEC. See https://www.sec.gov/edgar/searchedgar/cik.htm to search for a company’s CIK.

  • priorto (Union[str, datetime.datetime]) – Most recent report to consider. Must be in form ‘YYYYMMDD’ or valid datetime.datetime object.

  • filing_type (str) – Choose from list of valid filing types. Includes ‘10-Q’, ‘10-K’, ‘8-K’, ‘13-F’, ‘SD’.

Returns

None
