Filings¶

There are currently two supported ways to fetch filings. If you are interested in grabbing specific types of filings for a specific set of companies, it is suggested that you use secedgar.filings.Filing. If you are interested in fetching all filings from any given day, you should use secedgar.filings.DailyFilings.

Filings can be downloaded using secedgar.filings.Filing.

Supported filing types can be found at Filing Types

Filing¶

class secedgar.filings.Filing(cik_lookup, filing_type, start_date=None, end_date=datetime.datetime(2020, 5, 28, 18, 38, 58, 738011), client=None, **kwargs)¶

Base class for receiving EDGAR filings.

cik_lookup¶

Central Index Key (CIK) for company of interest.

Type

str

filing_type¶

Valid filing type enum.

Type

secedgar.filings.filing_types.FilingType

start_date¶

Date before which not to fetch reports. Stands for “date after.” Defaults to None (will fetch all filings before end_date).

Type

Union[str, datetime.datetime], optional

end_date¶

Date after which not to fetch reports. Stands for “date before.” Defaults to today.

Type

Union[str, datetime.datetime], optional

Keyword Arguments

count (int) – Number of filings to get per request. Passed to NetworkClient created on initialization if none given.

New in version 0.1.5.

Examples¶

Restrict the start and end dates by using the start_date and end_date arguments.

from secedgar.filings import FilingType, Filing
from datetime import datetime

filing = Filing(cik_lookup='aapl',
                filing_type=FilingType.FILING_10K,
                start_date=datetime(2015, 1, 1),
                end_date=datetime(2019, 1, 1))

If you would like to find all filings from some start_date until today, simply exclude end_date. The end date defaults to today’s date.

from secedgar.filings import FilingType, Filing
from datetime import datetime

filing = Filing(cik_lookup='aapl',
                filing_type=FilingType.FILING_10K,
                start_date=datetime(2015, 1, 1)) # end date defaults to today

You can also look up a specific filing type for multiple companies.

from secedgar.filings import FilingType, Filing
from datetime import datetime

filing = Filing(cik_lookup=['aapl', 'msft', 'fb'],
                filing_type=FilingType.FILING_10K,
                start_date=datetime(2015, 1, 1))

For a full list of the available filing types, please see secedgar.filings.FilingType.

Daily Filings¶

class secedgar.filings.DailyFilings(date, client=None)¶

Class for retrieving all daily filings from https://www.sec.gov/Archives/edgar/daily-index.

date¶

Date of daily filing to fetch.

Type

datetime.datetime

client¶

Client to use for fetching data. Defaults to secedgar.client.NetworkClient if none is given.

Type

secedgar.client._base.AbstractClient

Examples¶

If you wanted to download all filings from January 2, 2020, you could use the following code. Note that you should replace '/my_directory' with the desired directory of the filings.

from secedgar.filings import DailyFilings
from datetime import datetime

daily_filings = DailyFilings(date=datetime(2020, 1, 2))
daily_filings.save('/my_directory')
